# TEMPLATE DE CHECKPOINT HISTÓRICO — PREDIXAI BR

## Comando

    fechar
    checkpoint

## Estrutura obrigatória

1. Título sugerido do chat.
2. Objetivo do chat.
3. PTP principal.
4. Letra/subetapa atual.
5. Nome curto.
6. Etapa anterior.
7. O que foi concluído.
8. O que ficou pendente.
9. Roadmap visual completo.
10. Mini-PTPs criadas e motivo.
11. Scripts aplicados.
12. Relatórios TXT.
13. Markdown gerados em `docs/`.
14. Commits.
15. Pushes.
16. Problemas.
17. Decisões.
18. Riscos.
19. Estado do GitHub.
20. Próxima ação exata.
21. Comando de retomada.

## Flags

- ✅ concluída
- 🟨 em andamento
- 🟧 etapa atual
- ⏳ aguardando execução
- 🟦 auditada
- 🟪 pausada
- 🟥 falha
- ⛔ bloqueada
- ⬜ não iniciada
- 🚀 publicada
