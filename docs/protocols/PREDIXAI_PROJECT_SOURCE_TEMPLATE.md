## Regra de nome

Todo arquivo fonte ligado a uma PTP deve conter número e título da PTP.

Modelo:

    PREDIXAI_FONTE_PTP-113B_Primeiro_Ciclo_Operacional_Mobile_First.md

Título interno:

    # PTP-113B — Primeiro Ciclo Operacional Mobile-First Simulado Controlado

## Uso

Este arquivo deve ser baixado e adicionado manualmente como fonte do projeto no ChatGPT quando houver:

- mudança de governança;
- fechamento de PTP;
- nova política operacional;
- atualização de fluxo;
- novo padrão de skills;
- marco importante do projeto.
