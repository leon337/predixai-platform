# PredixAI BR — Roadmap Oficial

## Fase 0 — Fundação documental

- Criar documentação oficial
- Criar arquitetura
- Criar checklist da V1
- Criar contexto do projeto
- Criar repositório privado

## Fase 1 — Fundação técnica

- Estrutura de pastas
- Sistema de configuração
- Sistema de logs
- Contexto JSON
- Inicialização da aplicação
- PROJECT_RULES.md
- requirements.txt
- Preparação para arquivos sensíveis: .env, license.local.json e secrets.local.json

## Fase 2 — Vision

- Captura de tela
- Intervalo de 10 segundos
- Salvar screenshots
- Detectar áreas da tela
- Preparar OCR

## Fase 3 — Auditor

- Banco SQLite
- Registrar capturas
- Registrar dados detectados
- Registrar sinais
- Registrar motivo

## Fase 4 — Strategy

- Implementar Rebote Triplo
- Ler RSI 15, RSI 30 e RSI 60
- Gerar COMPRA, VENDA ou AGUARDAR
- Explicar motivo do sinal

## Fase 5 — AI Provider

- Configurar Gemini
- Analisar registros
- Gerar relatório simples
- Não operar
- Não tomar decisão financeira

## Fase 6 — Dashboard

- Interface simples
- Mostrar status
- Mostrar última captura
- Mostrar dados detectados
- Mostrar logs
- Mostrar sinal sugerido

## Fase 7 — Validação

- Testar leitura da tela
- Testar logs
- Testar auditoria
- Testar consistência
- Medir taxa de leitura correta

## Após a V1 — V2 futura

- Simulação em demo
- Validação de estratégia
- Melhor leitura visual
- Replay
- Backtest
