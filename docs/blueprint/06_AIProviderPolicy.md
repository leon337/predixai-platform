# PredixAI BR — Política de IA v1.0

## 1. Papel da IA na V1

A IA será analista, não operadora.

## 2. Provedor inicial

Google Gemini.

## 3. Funções permitidas

- resumir dados do Auditor;
- gerar relatório diário;
- detectar inconsistências;
- explicar sinais;
- sugerir melhorias;
- analisar padrões históricos.

## 4. Funções proibidas na V1

- clicar;
- operar;
- executar ordem;
- alterar estratégia sozinha;
- decidir entrada real;
- aumentar risco automaticamente.

## 5. AI Provider

O sistema deve ser construído com camada de abstração.

O Core deve poder trocar de IA no futuro sem reescrever os demais módulos.

## 6. Provedores futuros

- DeepSeek
- OpenAI
- OpenRouter
- Ollama
- IA local

## 7. Segurança

A chave da API deve ser local e ignorada pelo Git.

## 8. Custo

A V1 deve usar preferencialmente camada gratuita para testes.
