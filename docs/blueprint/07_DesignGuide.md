# PredixAI BR — Guia de Design v1.0

## 1. Direção visual

Visual escuro, tecnológico, limpo e profissional.

## 2. Paleta inicial

- Fundo: preto / azul escuro
- Destaque: verde neon
- Alerta: amarelo
- Erro: vermelho
- Texto: branco e cinza claro

## 3. V1 Dashboard

A tela deve comunicar:

**O robô está observando, não operando.**

## 4. Blocos principais

- Status do sistema
- Tela capturada
- Dados detectados
- RSI 15, 30 e 60
- Sinal sugerido
- Motivo do sinal
- Últimas leituras
- Logs

## 5. O que não mostrar na V1

- Botão de executar operação
- Lucro real
- Promessa de ganho
- Marketplace
- Estratégias extras
- Execução automática

## 6. Tom da interface

Profissional, claro e transparente.

O usuário precisa entender exatamente o que o sistema viu e por que sugeriu um sinal.
